# Ecommerce WebApi 

#### Ecommerce API is built using ASP.NET Core 8 with Clean Architecture and The Best Practices.

## Technologies / Principles used : 

- Asp.net core webapi 8
- Entity framewrok 8
- SqlServer
- Identity/Jwt Tokens and Refresh Tokens for Auth
- AutoMapper
- AspNetCoreRateLimit (Rate Limit)
- Testing using `xunit`
- Clean Architecture (Onion Architecture)
- Repository Pattern & Separation of Concerns (SoC)
- Dependency Injection (IoC)
- Unit of Work pattern (UoW)
- Options Pattern
